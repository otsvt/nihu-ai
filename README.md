# НИХУ-AI

Сатирический адаптивный AI-SaaS лендинг на русском языке. Проект работает как клиентское SPA без backend и внешних API.

## Запуск

```bash
npm install
npm run dev
```

## Проверки и сборка

```bash
npm run lint
npm run build
npm run preview
```

## Маршруты

- `/` — основной лендинг с псевдодашбордом, генератором успеха, лентой активности, тарифами и FAQ.
- `/about` — история компании, миссия, ценности и команда.
- `/donate` — шуточные донаты без настоящей оплаты.

Состояние cookie-баннера хранится локально в браузере. Все метрики генерируются на клиенте.
